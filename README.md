Ticket bot for Discord made by me

Features:
-.env (managing bot, ids, image and emojis)
-Open ticket
-Close ticket
-Public Embed with title, desc, title and a button with categories with title, desc and emoji then open ticket on separated catogries
-Commands for change stuffs on public and private embeds

Commands:

/panelset - change stuffs on public panel
/panelshow - show a preview of public panel
/ticketset - change stuffs on private panel
/ticketsetup - send the public panel on current channel ure typing
/ticketshow - show a preview of private panel
