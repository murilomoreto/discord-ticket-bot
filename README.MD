Ticket bot for Discord made by me

Features:
    -env (managing bot, ids, image and emojis) <br>
    -Open ticket <br>
-Close ticket <br>
-Reopen ticket <br>
-Delete ticket <br>
-Public Embed with title, desc, title and a button with categories <br> with title, desc and emoji then open ticket on separated catogries
-Commands for change stuffs on public and private embeds <br>

Commands: <br>
/panelpost - public the panel on current solicited channel on .env <br>
/panelset - change stuffs on public panel <br>
/panelshow - show a preview of public panel <br>
/ticketset - change stuffs on private panel <br>
/ticketsetup - send the public panel on current channel ure typing <br>
/ticketshow - show a preview of private panel <br>

How to use: <br>

-Install Node.js <br>
-Open cmd and type "cd" and paste folder path then "npm i" <br>
-Set up .env <br>
